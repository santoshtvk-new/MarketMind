# MarketMind
## How to Run
1. Clone the repository or copy the source code (app.py and requirements.txt).
2. Install dependencies (if you haven't already): `pip install -r requirements.txt`
3. Run the application: `streamlit run app.py`
4. The application will open automatically in your default web browser.

## Usage
- Sidebar: Select any ticker symbol (e.g., NVDA, GOOGL) and select a time period.
- Charts Tab: View price action and moving averages.
- Financials Tab: See key statistics like Market Cap, PE Ratio, and Company Summary.
- News Tab: Read the latest headlines and see their AI-generated sentiment analysis.
